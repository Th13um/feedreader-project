## Tests

**RSS Feeds:**
* Feeds are Defined
* Feed URL is not empty
* Feed names are not empty

**The Menu:**
* The menu is hidden by default
* When clicked the menu changes visibility

**Initial Entries:**
* Feed container has at least 1 entry

**New Feed Selection:**
* Feed content changes when new feed is selected
